# MindDiary

MindDiary는 일상의 일기를 시각적인 "생각의 은하계(Mind Galaxy)"로 변환하는 인터랙티브 웹 애플리케이션입니다. 텍스트를 분석하여 키워드와 그들 간의 관계를 추출하고, 생각과 감정의 흐름을 동적으로 탐색할 수 있는 지도를 생성합니다.

## 주요 기능

*   **생각의 지도 & 마인드 갤럭시**: 생각의 흐름을 방향성 그래프로 시각화하며, 3D/2D 인터랙티브 은하계로 진화합니다.
*   **감정 분석**: 키워드의 감정을 자동으로 분석합니다.
    *   **긍정**: 파란색 노드
    *   **부정**: 빨간색 노드
    *   **중립**: 랜덤 파스텔 톤 노드
*   **동적 시각화**:
    *   **노드 크기 조절**: 키워드 빈도와 연결성에 따라 노드 크기가 커집니다.
    *   **입자 흐름 효과**: 연결선을 따라 흐르는 애니메이션 입자로 의식의 흐름과 감정 전이를 표현합니다.
    *   **물리 엔진**: 드래그 앤 드롭을 지원하며 노드들이 동적으로 상호작용합니다.
*   **인터랙티브 UI**:
    *   2D와 3D 뷰 간 전환 가능
    *   노드만, 텍스트만, 또는 모두 표시하는 필터 기능
    *   긴 일기 작성을 위한 조절 가능한 입력 영역

## 기술 스택

*   **백엔드**: FastAPI (Python)
*   **프론트엔드**: HTML, CSS, JavaScript (force-graph)
*   **자연어 처리**: KonlPy (한국어 자연어 처리)
*   **그래프 처리**: NetworkX

## 설치 방법

1.  **사전 요구사항**:
    *   Python 3.8 이상
    *   Java (JDK) - `konlpy` 실행에 필요

2.  **저장소 클론**:
    ```bash
    git clone <repository-url>
    cd MindDiary
    ```

3.  **의존성 설치**:
    ```bash
    pip install -r requirements.txt
    ```
    *포함된 의존성: `fastapi`, `uvicorn[standard]`, `konlpy`, `networkx`, `matplotlib`, `jinja2`*

4.  **환경 설정**:
    *   시스템에 맞게 `JAVA_HOME` 환경 변수가 올바르게 설정되어 있는지 확인하세요.

## 사용 방법

1.  **서버 실행**:
    ```bash
    uvicorn main:app --reload
    ```

2.  **애플리케이션 접속**:
    *   웹 브라우저를 열고 `http://localhost:8000`으로 이동하세요.

3.  **마인드 갤럭시 생성**:
    *   입력창에 일기를 작성하세요.
    *   "분석하기" 버튼을 클릭하여 생각의 지도를 생성하세요.
